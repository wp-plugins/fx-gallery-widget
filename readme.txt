Simple widget for displaying gallery images of current page or post, or from a specific post or page set by ID

== Description ==

FX Gallery Widget adds a widget for displaying gallery for attached images to current page or post, or to a specific post or page by ID. It is used to show images in widget area if they are linked to a page or post, they don't have to be inserted in page as images. Widget has setting for number of images to display, also displaying order may be set to random.

= Features =
* Show images from currently displayed page or post
* Add as many widgets as you want, wherever you want
* Select a number of images
* Select any registered size in WordPress
* Display the images in ascending, descending, or random order
* Link the images to the original file, post, anchor in the post, attachment page, or nothing
* Add a prefix to the link and image title (appears as a tooltip)
* Use a rel attribute for the link - great for lightboxes

== Installation ==

FX Gallery Widget is easily installed automatically via the Plugins tab in your blog administration panel.

= Manual Installation =

1. Upload the `fx-gallery-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance &rarr; Widgets to setup FX Gallery Widget

== Frequently Asked Questions ==

= How do I get the ID for the post or page? =
You may install "WP Show IDs" plugin, then all ID's will be shown in administration pages.
